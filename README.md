# Site Scraper

A robust Python script designed to crawl web pages and efficiently extract/download all linked files and assets from a specified target URL. 

### Tech Stack
* **Language:** Python 3.x
* **Core Libraries:** `BeautifulSoup4`, `requests`, `os`

### How to Use
1. Clone the repository.
2. Install required packages: `pip install -r requirements.txt`
3. Execute the script with your target URL: `python scraper.py --url https://example.com`

*Disclaimer: Ensure you have the right to scrape the target website and respect their `robots.txt` policies.*
